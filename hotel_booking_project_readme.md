# Hotel Booking Data Analysis

## Overview
This project focuses on analyzing hotel booking data to uncover trends, customer behavior, reservation patterns, and business insights. The analysis was performed using Python and popular data analysis libraries.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset
The dataset used in this project contains hotel booking information such as:
- Guest details
- Reservation status
- Market segments
- Average Daily Rate (ADR)
- Customer types
- Room preferences
- Monthly booking trends

Dataset File:
```txt
hotel_booking.csv
```

---

## Project Objectives
- Analyze monthly guest booking trends
- Study Average Daily Rate (ADR)
- Identify the most common market segments
- Calculate repeated guest percentages
- Find the most reserved room types
- Analyze customer types and reservation patterns

---

## Key Analysis Performed
### 1. Monthly Guest Count
Tracked the number of guests visiting each month to identify seasonal booking trends.

### 2. Average Daily Rate (ADR)
Analyzed hotel pricing trends and customer spending patterns.

### 3. Market Segment Analysis
Studied which market segments contributed the most bookings.

### 4. Repeated Guest Percentage
Measured customer retention and repeat visits.

### 5. Most Reserved Room Type
Identified the room categories with the highest demand.

### 6. Customer Type Analysis
Analyzed different customer groups and their booking behavior.

### 7. Reservation Percentage
Calculated reservation distribution and booking patterns.

## How to Run the Project
1. Clone the repository
2. Install required libraries:

```bash

```

3. Open the notebook:

```bash
jupyter notebook
```

4. Run all cells in `Hotel_booking.ipynb`
